# Cómo ejecutar el proyecto

```bash
git clone https://github.com/TUUSUARIO/TiendaJama.git

cd TiendaJama

python -m venv venv

venv\Scripts\activate

pip install django

python manage.py migrate

python manage.py runserver
```
